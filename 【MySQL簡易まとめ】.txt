基本操作

　　ログイン(パス無し)
　　　　mysql -u [ユーザー名]

　　存在するデータベースの表示
　　　　show databases;

　　データベースの作成
　　　　create database [データベース名];

　　データベースの削除
　　　　drop database [データベース名];

　　利用するデータベースの選択
　　　　use [データベース名]
